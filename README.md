## Extensão Softcom para Suporte

Esta extensão foi criada para facilitar e agilizar processos repetitivos no suporte da Softcom em Patos.

### Funcionalidades

- Abertura rápida de nova ocorrência e visualização de informações do cliente atual do chat para a área Partner.
- Pesquisa rápida do cliente atual do chat por nome.
- Inserção automática de valores padrão em formulários de criação de ocorrências na área Partner, incluindo configurações para SPED e encaminhamento ao Apoio Técnico da matriz.

### Instalação

1. Abra o Chrome ou navegador semelhante.
2. Vá em **Extensões** → **Modo Desenvolvedor**.
3. Clique em **Carregar sem compactação** e selecione a pasta desse projeto.
4. Atualize qualquer página que você já tenha aberto (do cliente satisfeito ou área partner) para que a extensão seja carregada corretamente.
