## Extensão Softcom para Suporte

Esta extensão foi criada para facilitar e agilizar tarefas repetitivas no suporte da Softcom em Patos.

### Funcionalidades

- Pesquisa rápida do cliente atual por nome na área Partner, a partir do chat "Cliente Satisfeito".
- Inserção automática de valores padrão em formulários de criação de ocorrência (SPED e envio para Apoio Técnico da matriz).

### Instalação

1. Abra o Chrome.
2. Vá em **Extensões** → **Modo Desenvolvedor**.
3. Clique em **Carregar sem compactação** e selecione esta pasta.
